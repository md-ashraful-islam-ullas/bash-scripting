# Bash Variables & Data Types – Practice Exercises

This folder contains a collection of beginner-level Bash scripting exercises focused on variables, strings, and basic data manipulation.


## Scripts Included

- `arithmetic_operatin.sh` – Performs basic arithmetic operations like addition, subtraction, multiplication, and division using Bash
- `floating_point_arithmetic.sh` – Handles floating-point calculations using external tools like `bc`
- `string_length.sh` – Finds the length of a string using `${#var}` parameter expansion
- `string_concatinate.sh` – Joins multiple strings together into one combined output
- `uppercase_lowercase_conversion.sh` – Converts strings between uppercase and lowercase using Bash expansion and/or `tr`
- `subtracting_extraction.sh` – Extracts a portion of a string using substring slicing `${var:start:length}`
- `default_variable_values.sh` – Demonstrates default values using `${var:-default}` when variables are unset or empty

## Purpose

- Variables and data types in Bash
- String manipulation techniques
- Basic arithmetic operations
- Command-line parameter handling
- Default value handling
- Text processing using built-in Bash features and utilities
