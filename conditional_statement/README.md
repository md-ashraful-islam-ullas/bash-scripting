# Bash Conditional Statements – Practice Exercises

This folder contains a collection of beginner-level Bash scripting exercises focused on conditional statements, logical operations, and decision-making in shell scripts.

## Scripts Included

- `case_statement_menu.sh` – A menu-driven program using `case` statements to perform different system tasks
- `even_odd.sh` – Checks whether a given number is even or odd using conditional logic
- `file_type_check.sh` – Identifies whether a path is a file, directory, or other type
- `grade_calculator.sh` – Calculates grades based on user input marks using `if-elif-else`
- `leap_year_check.sh` – Determines whether a given year is a leap year using logical conditions
- `number_comparsion.sh` – Compares two numbers and determines which is greater or if they are equal
- `pass_strength_checker.sh` – Evaluates password strength based on defined conditions

## Purpose

- Understanding `if`, `elif`, and `else` statements 
- Working with logical operators (`&&`, `||`) 
- Using `case` statements for menu-driven programs 
- Performing numeric comparisons 
- Handling user input in Bash scripts 
- Building basic decision-making logic in shell scripting
